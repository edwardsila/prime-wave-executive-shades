🚀 HOW TO GET STARTED
====================

OPTION 1: Automated Setup (Recommended)
Linux/Mac:
  bash install.sh

Windows:
  install.bat

OPTION 2: Manual Setup
Backend:
  cd backend
  npm install
  cp .env.example .env
  npm run dev

Frontend (in new terminal):
  cd frontend
  npm install
  npm start

═══════════════════════════════════════════════════════════════════════════

⚙️ INITIAL CONFIGURATION
========================

1. Update MongoDB URI in backend/.env
   MONGODB_URI=your_connection_string

2. Create admin account:
   curl -X POST http://localhost:5000/api/admin/register \
     -H "Content-Type: application/json" \
     -d '{"username":"admin","email":"admin@example.com","password":"yourpass"}'

3. Update WhatsApp number in:
   frontend/src/components/WhatsAppBubble.js

4. Replace hero images in:
   frontend/src/components/HeroSection.js

5. Customize colors (search & replace in CSS):
   #667eea -> your_color_1
   #764ba2 -> your_color_2
